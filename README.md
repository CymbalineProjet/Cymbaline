<h4>Create a new repository on the command line</h4>

touch README.md<br/>
git init<br/>
git add README.md<br/>
git commit -m "first commit"<br/>
git remote add origin https://github.com/DKsProject/DKs-Fram.git<br/>
git push -u origin master<br/>

git remote add origin https://github.com/DKsProject/DKs-Fram.git<br/>
